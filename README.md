## 💡 使用技巧

### 考点速查
1. 首页选择年级
2. 点击章节卡片查看详情
3. 查看考点内容、关键提示、常见错误
4. 点击"练习本章题目"进行针对性练习

### 公式速记
1. 点击分类筛选公式
2. 使用搜索框快速查找
3. 重点关注"口诀"帮助记忆

### 刷题模式
- **随机练习**：每次 10 道随机题目
- **章节练习**：从章节详情页进入，针对性巩固
- 答错自动加入错题本
- 答题结束显示正确率和用时

### 错题本
- 所有错题自动保存（浏览器本地存储）
- 可按章节筛选
- 点击"已掌握"移除错题
- 支持清空错题本

## 🛠️ 扩展与维护

### 添加新题目
编辑 `js/quiz-data.js`，按格式添加：

```javascript
{
  id: 'q-g7-1-c2-006',
  chapterId: 'g7-1-c2',  // 对应章节 ID
  type: 'choice',
  difficulty: 2,
  question: '题目内容',
  options: ['A. 选项 1', 'B. 选项 2', 'C. 选项 3', 'D. 选项 4'],
  answer: 'A',
  analysis: '详细解析...'
}
```

### 添加新考点
编辑 `js/data.js`，在对应章节的 `examPoints` 数组中添加。

### 添加新公式
编辑 `js/data.js`，在 `FORMULA_DATA.categories` 对应分类中添加。
